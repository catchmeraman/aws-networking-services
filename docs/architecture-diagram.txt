Components:
- EKS Cluster: unicorn-cluster
- ALB: Internet-facing with SSL termination
- ACM Certificate: eksawsweek2.cloudopsinsights.com
- ECR Repository: Container image storage
- Kubernetes Service: ClusterIP load balancing
- Pods: 2 replicas for high availability
