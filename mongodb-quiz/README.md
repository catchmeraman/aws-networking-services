# MongoDB Atlas Quiz Application

A comprehensive quiz application about MongoDB Atlas with user authentication, scoring system, and admin dashboard.

## Features

### User Authentication
- User registration and login system
- Secure password-based authentication
- Session management

### Quiz System
- 10 multiple-choice questions about MongoDB Atlas
- Real-time scoring
- Progress tracking
- Immediate results display

### Scoring & Analytics
- Personal best score tracking
- Total attempts counter
- Last played timestamp
- Persistent data storage using localStorage

### Leaderboard Dashboard
- Top 10 users ranking
- Sortable by best scores
- User statistics display
- Real-time updates

### Admin Features
- Admin login (username: admin, password: admin123)
- Reset all scores functionality
- User management capabilities
- Protected admin panel

## Default Admin Credentials
- **Username**: admin
- **Password**: admin123

## How to Use

1. **Registration**: New users can register with username/password
2. **Login**: Existing users login with credentials
3. **Take Quiz**: Answer 10 MongoDB Atlas questions
4. **View Results**: See score and percentage
5. **Dashboard**: Check leaderboard and rankings
6. **Admin**: Reset scores (admin only)

## Quiz Topics Covered

- MongoDB Atlas basics
- Deployment options
- Replica sets and sharding
- Atlas features (Search, Charts, Data Lake)
- MongoDB fundamentals
- Query language and drivers
- Performance and scaling

## Technical Details

- Pure HTML/CSS/JavaScript (no external dependencies)
- Client-side data persistence with localStorage
- Responsive design for all devices
- MongoDB Atlas themed styling
- Secure admin functionality

## Deployment

Simply serve the `index.html` file from any web server or hosting platform.
